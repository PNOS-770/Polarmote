跨平台终端模拟器与远程管理工具，支持 **Android** 和 **Windows**。

## 功能

- **连接管理** — SSH（密码/密钥/Agent）、本地终端、串口、Telnet；ProxyJump 跳板、SOCKS5 代理；会话树分组、搜索、筛选、排序、常用主机固定
- **终端** — 多 Stage 标签页管理、分屏、广播输入；终端搜索（正则）、块选择；字体/字号/光标样式可配置
- **文件传输** — SFTP（Rust 原生加速），队列、断点续传、自动重试；远程文件树浏览与在线编辑
- **脚本自动化** — 脚本管理（文件夹分组、参数变量）
- **端口转发** — 本地/远程/SOCKS5 动态转发，模板保存自动启动
- **服务器监控** — 实时 CPU、内存、磁盘指标仪表盘
- **快捷键** — 全快捷键自由重绑定，多套预设
- **界面** — 中文 / English 双语；终端背景图片、透明度设置
