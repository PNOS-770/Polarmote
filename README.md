# Polarmote

跨平台终端模拟器与远程管理工具，基于 Flutter 构建。支持 SSH、本地 Shell、串口、Telnet，集成文件传输、脚本自动化、端口转发。

[下载](https://github.com/PNOS-770/Polarmote/releases)

---

- **多协议** — SSH（密码/密钥/Agent）、本地终端、串口、Telnet
- **文件传输** — SFTP 传输队列，远程文件浏览与编辑
- **脚本自动化** — 脚本管理、工作流、批量执行、定时调度、触发器
- **端口转发** — 本地/远程/SOCKS5，模板化管理
- **分屏终端** — 多布局、广播输入、搜索、块选择
- **服务器监控** — 实时 CPU、内存指标
- **AI 助手** — 内置对话面板
- **自定义快捷键** — 所有快捷键可重新绑定
- **多语言** — 中文 / English

Windows / Linux / macOS / Android / iOS
