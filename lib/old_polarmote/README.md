# Asmote

**Asmote** 是一款跨平台终端模拟器与远程管理工具，基于 Flutter 构建。支持 SSH、本地 Shell、串口、Telnet 等多种连接协议，集成分屏终端、文件传输、脚本自动化、端口转发等功能，适用于开发者和运维人员的日常服务器管理。

## 核心特性

### 连接管理

- **多协议支持**：SSH（密码/密钥/Agent 认证）、本地终端（PowerShell、CMD、WSL、bash）、串口、Telnet
- **跳板机与代理**：SSH ProxyJump、SOCKS5 代理
- **会话树**：分组管理主机，支持搜索、筛选、排序、固定常用主机
- **快速连接**：一键发起临时连接
- **自动重连**：断线后自动尝试恢复

### 分屏终端

- **多布局**：水平分割、垂直分割、网格布局，支持保存/加载布局模板
- **广播输入**：同时向所有面板发送键盘输入
- **终端搜索**：支持正则表达式搜索终端输出
- **块选择**：行选择与块选择模式一键切换
- **外观定制**：字体/字号/行高、配色方案、背景图片（支持每面板独立配置）

### 文件传输与浏览

- **SFTP 传输**：基于 Rust 原生加速引擎，支持上传/下载/批量队列
- **传输控制**：暂停、恢复、取消、自动重试
- **远程文件管理**：浏览远程目录树，新建/重命名/删除文件与文件夹
- **文件查看与编辑**：在线预览和编辑远程文件

### 脚本自动化

- **脚本管理**：创建/编辑/删除脚本，支持文件夹分组和参数变量
- **工作流**：将多个脚本编排为顺序执行序列
- **批量模板**：预配置目标主机，一键批量执行
- **定时调度**：Cron 表达式设置定时执行
- **触发器**：连接成功或提交命令时自动执行脚本

### 端口转发

- **本地转发**：将本地端口转发到远程目标
- **远程转发**：将远程端口转发到本地
- **SOCKS 代理**：动态端口转发
- **模板管理**：保存为可复用模板，支持连接时自动启动

### 服务器监控

- **服务器仪表盘**：概览所有主机的连接状态
- **资源监控**：实时 CPU、内存等指标展示

### AI 助手

- **AI 对话**：内置 AI 聊天面板，可解释终端输出、协助排查问题
- **预设配置**：可自定义 AI 预设与 API 密钥

### 界面与个性化

- **自定义快捷键**：所有快捷键均可重新绑定，内置多套预设
- **多语言**：支持中文和英文
- **深色/浅色模式**：跟随系统或手动切换

### 安全

- **加密存储**：密码、私钥口令等敏感信息加密保存
- **主机密钥验证**：首次连接提示确认，密钥变更告警
- **SSH Agent**：支持系统 SSH Agent 认证

### 跨平台

Windows / Linux / macOS / Android / iOS

## 默认快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+C` / `Ctrl+C` | 复制 |
| `Ctrl+V` / `Shift+Insert` | 粘贴 |
| `Ctrl+A` | 全选 |
| `Ctrl+F` | 查找 |
| `Alt+B` | 切换块选择 |
| `Ctrl+D` | 分屏 |
| `Ctrl+Alt+Enter` | 最大化 / 还原窗格 |
| `Ctrl+Alt+B` | 切换输入广播 |
| `Ctrl+Alt+方向键` | 切换窗格 |

所有快捷键均可在 **设置 → 快捷键** 中自定义。
